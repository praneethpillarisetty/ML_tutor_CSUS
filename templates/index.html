Student Progress Log API

Manage student progress logs for machine learning courses

API Documentation

POST /log

Add a new student progress entry to the log.

Requires: email, student_id, week, exercise, status, feedback
GET /logs

Retrieve all logs with optional filtering.

Optional params: email, student_id, week
DELETE /logs

Clear all logs (requires secret key).

Requires: key parameter
